# Principal Component Analysis (PCA) on Wine Dataset

## Objective
To reduce the dimensionality of the Wine Dataset using Principal Component Analysis (PCA) and visualize the transformed data in two dimensions.

## Tools Used
- Python
- Google Colab / Jupyter Notebook
- Pandas
- NumPy
- Matplotlib
- Scikit-learn

## Dataset
Wine Dataset (Built-in Scikit-learn Dataset)

## Methodology
1. Load the Wine Dataset.
2. Standardize the features.
3. Apply PCA for dimensionality reduction.
4. Transform data into two principal components.
5. Analyze explained variance ratio.
6. Visualize transformed data in 2D.

## Deliverables
- Transformed Dataset
- Variance Plot
- 2D PCA Visualization

## Results
PCA successfully reduced the dataset dimensions while preserving most of the important information. The first two principal components explained a significant portion of the total variance.

## Conclusion
Principal Component Analysis is an effective dimensionality reduction technique that simplifies high-dimensional datasets while retaining essential information. It improves visualization and reduces computational complexity.
